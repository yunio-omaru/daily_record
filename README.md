# 🌿 Life Log Note (スキンケア・食事・運動記録)

毎日のスキンケア、食事、運動の記録を「朝・昼・晩」の時間帯ごとにスマートに管理できる、自分専用のライフログページです。
データベース不要でブラウザのみで完結し、データはローカルストレージに安全に自動保存されます。

## 🔗 公開ページURL
https://yunio-omaru.github.io/daily_record/
---

## ✨ 主な機能とこだわりポイント

1. **カレンダー連携 ＆ インジケーター**
   - 日付を選択してその日の記録をピンポイントに確認・編集できます。
   - 何かしらの記録がある日には、カレンダー上に **●（オレンジの点）** が自動で表示されるため、継続のモチベーションに繋がります。

2. **朝・昼・晩の時間帯管理**
   - 「朝」「昼」「晩」のタブを切り替えることで、それぞれのタイミングに合わせた詳細なメモが可能です。

3. **魔法の「自動行追加」システム**
   - メモ欄に1行入力すると、**自動で次の行の入力欄が下にフワッと出現**します。最初から大量の空欄で画面を圧迫せず、書きたい分だけスムーズに入力できます。

4. **賢い「予測候補（サジェスト）」機能**
   - 一度入力して保存したワード（化粧品名、メニュー、使用量など）は、次回入力時に**予測候補として自動でポップアップ**します。毎日のルーティン入力を極限までラクにします。
   - 項目（列）ごとに独立して記憶するため、候補が混ざることもありません。

---

## 🛠 使い方

1. **日付と時間帯を選ぶ**
   - カレンダーから記録したい日をクリックし、[朝] [昼] [晩] のタブを選択します。
2. **登録する**
   - 右上の `[+登録]` ボタンを押すと編集モードになります。
   - 文字を入力すると自動で行が増えます。
3. **保存する**
   - 下部の `[保存する]` ボタンを押すと、データが記憶され閲覧画面に戻ります。

---

## 🔒 技術的な仕様
- **フロントエンド:** HTML5 / CSS3 / JavaScript (Vanilla JS)
- **データ保存:** 外部のサーバーには一切データを送信せず、ブラウザの `LocalStorage` に保存されます。そのため、プライバシーが守られた状態で安心して利用できます。
